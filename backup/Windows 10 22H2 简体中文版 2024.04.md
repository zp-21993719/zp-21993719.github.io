- 采用Windows 10 Version 22H2简体中文商业版制作，联网自动数字激活，补丁升级至2024年04月09日。
- 移除OneDrive；卸载App应用：Cortana、人脉、Skype。
- 集成VC++ 2005至2022运行库。
- 集成DirectX 9.0c 运行库。
- 启用.NET Framework 3.5 (包括 .NET 2.0 和 3.0)。
- 启用 SMB 1.0/CIFS 文件共享支持。
- 更新Microsoft Edge v124.0.2478.51。
- 集成WinRAR v7.00 简体中文版。
- 集成Adobe Flash Player v34.0.0.308 去广告版。
- 显示桌面图标：用户的文件、计算机、网络、回收站、Internet Explorer。
- 安装StartIsBack++ v2.9.20经典开始菜单(可选)。
- 注入Intel RST VMD快速存储驱动 v20.0.0.1037。
- 禁用保留存储。
- 关闭 Microsoft Defender 防病毒。
- 关闭 Windows Defender SmartScreen。
- 关闭程序兼容性助手。
- 关闭休眠。
- 关闭打开程序的安全警告。

- 添加政府公文必备字体：仿宋_GB2312，楷体_GB2312，方正小标宋简体。
- 启用无线服务。
- 无人值守安装，默认Administrator登录，密码为空。

#### ★ 安装说明：
- 常规：优盘安装IT天空的优启通、微PE、系统总裁的U盘魔术师(USM)、USBOS等PE，优盘启动进入WinPE环境：使用Easy Image X(EIX)、一键备份还原(SGI)、CGI、WinNTSetup等工具安装即可。
- 高级：优盘安装Ventoy，把优启通等PE生成ISO，然后把PE的ISO文件和系统的ISO镜像复制到优盘，就可以想用哪个PE用哪个PE，而且可以像光盘一样直接安装ISO。

> 特别说明：如果用优盘安装系统，安装结束后，一定要拔掉优盘，不然会出现盘符错乱。
