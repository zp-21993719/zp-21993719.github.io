### Windows 10 企业版 LTSC 2019 简体中文版 2024.02

**Windows 10 企业版 LTSC：**
- Windows 10 企业版 LTSC 2021 中的功能相当于 Windows 10 版本 21H2。
- Windows 10 企业版 LTSC 2019 中的功能相当于 Windows 10 版本 1809。
- Windows 10 企业版 LTSC 2016 中的功能相当于 Windows 10 版本 1607。

#### ★ 更新说明：
1. 补丁升级至2024年02月13日。
2. 更新Microsoft Visual C++ 2015-2022 Redistributable – 14.38.33135.0。
3. 更新WinRAR v6.24 简体中文版。
4. 更新Adobe Flash Player v34.0.0.305 去广告版。

#### ★ 系统特色：
- 采用Windows 10 Enterprise LTSC 2019制作，联网自动数字激活，升级补丁至2024年02月13日。
- 集成VC++ 2005/2008/2010/2012/2013/2022运行库。
- 集成DirectX 9.0c 运行库。
- 启用.NET Framework 3.5 (包括 .NET 2.0 和 3.0)。
- 升级Microsoft .NET Framework 4.8 正式版。
- 启用 SMB 1.0/CIFS 文件共享支持。
- 集成WinRAR v6.24 简体中文版。
- 集成Adobe Flash Player v34.0.0.305 去广告版。
- 显示桌面图标：用户的文件、计算机、网络、回收站、Internet Explorer。
- 安装StartIsBack++ v2.9.20经典开始菜单(可选)。
- 注入Intel RST VMD快速存储驱动 v19.5.3.1050。
- 关闭 Microsoft Defender 防病毒。
- 关闭 Windows Defender SmartScreen。
- 关闭程序兼容性助手。
- 关闭休眠。
- 关闭打开程序的安全警告。
- 去除新建快捷方式字样。
- 删除右键菜单“新建”中不常用菜单。
- 右键添加“管理员取得所有权”。
- 右键菜单添加“用记事本打开”。
- 右键属性中增加文件校验功能。
- 添加政府公文必备字体：仿宋_GB2312，楷体_GB2312，方正小标宋简体。
- 启用无线服务。
- 无人值守安装，默认Administrator登录，密码为空。

#### ★ 安装说明：
- 常规：优盘安装IT天空的优启通、微PE、系统总裁的U盘魔术师(USM)、USBOS等PE，优盘启动进入WinPE环境：使用Easy Image X(EIX)、一键备份还原(SGI)、CGI、WinNTSetup等工具安装即可。
- 高级：优盘安装Ventoy，把优启通等PE生成ISO，然后把PE的ISO文件和系统的ISO镜像复制到优盘，就可以想用哪个PE用哪个PE，而且可以像光盘一样直接安装ISO。

> 特别说明：如果用优盘安装系统，安装结束后，一定要拔掉优盘，不然会出现盘符错乱。
